# 关于创作的思考

## 人类与大模型创作的区别

人类创作是一个闭环：写出去的东西，最终会流回来重新塑造写作者。

而当前的大模型创作，是一个开环：吐出来就结束了，没有真正的“自我”被更新。

这跟“有没有意识”无关，而是在说有没有那种被时间、反馈、历史作品层层叠出来的稳定演化轨迹。

哪怕这个轨迹很慢、很微小，它也是“作者性”的来源。

## 什么是作者性？

作者性不是一次写得好不好的能力，而是一段时间内，你身上那些稳定的、持续的东西。

比如：

- 你总是用那种冷冰冰的短句（风格连续性）

- 你翻来覆去都在写“记忆与遗忘”这个主题（主题执念）

- 读者说这里太晦涩，你下一本就会微调，但不会完全丢掉自己（反馈修正）

- 别人一看就知道是你写的（身份稳定性）

- 你的作品之间形成一种内在张力，好像自己在对话（历史张力）

这些加起来，才构成了“你作为一个作者”的身份。它是一个跨时间积累出来的稳定状态。

## 创作的本质是什么？

创作是在特定审美场中的涌现。

我们把创作定义为作者、作品、观众构成的动态反馈网络：文化环境不是静态背景，而是长期演化出的约束势场；

创作则是在这一势场中发生的局部低熵信息生成。

- 创作系统 = 作者、作品、观众构成的动态反馈网络。
- 文化环境 = 网络长期演化后形成的统计性约束势场。
- 创作行为 = 个体在约束势场中的局部低熵信息结构生成。
- 作品传播 = 信息结构在群体认知中的再编码与变异扩散。
- 文化演化 = 不同信息结构在社会注意力与时间中的选择过程。

（按此严格定义，当前大模型生成文本的行为，更接近从文化约束场中进行条件采样，而非完整意义上的创作。）

## 大模型的一些表现

大模型频繁使用“不是……而是……”句式，因为它是训练数据中表达转折、对比的高概率句式结构。

直接禁止作用不大，模型仍会寻求其他连接词完成同样的语义功能，往往生成更别扭的句子（如“你思考创作并非实质，真相在于……”）。

这恰恰说明，人类本身就习惯用这种结构来纠正误解和强调本质。

## 大模型的问题出在哪儿？

现在的大模型创作是这样的：

文化约束场 → 作品

就是：模型吸收了网上所有文本（约束场），然后你给它一个提示，它输出一段文字。作品生成完了就完了，没有后续。

它缺少了那个关键的闭环：

- 它没有一个随时间演化的“作者状态”

- 作品不会反过来持久地塑造模型本身

- 它不会因为今天写了一个差评如潮的故事，而从此改变自己的“人格”或写作倾向

所以，它更像是从文化的平均概率里直接采样，而不是一个有内在历史、能被反馈持续改变的创作者。

## 控制论思路与复杂科学思路

单纯的控制论（给定目标，负反馈纠偏）在面对大模型这个复杂系统时，已经逐渐暴露出局限性。

就像用手按住水下冒上来的气泡，用手盖住这一块，气泡还是会从其他地方冒出来。

那么，真正的道路在哪里呢？

我们或许应该从复杂科学中寻找答案。

- 把模型视为动力学系统，把生成视为吸引子之间的相空间流动。

- 用涌现、相变、自组织临界性等概念重新描述模型行为，建立新的分析语言。

- 将研究目标从“纠错”转向“探索和重塑系统的可能性景观”。
